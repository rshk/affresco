Introducing Affresco.

Affresco was born as a fork of Twitter Bootstrap.

SCSS instead of LESS.

I like the power & clarity of SCSS over LESS, plus this way we can use the awesome Compass library.

Easier to extend.

One of the main goals of the fork was to made the library easier to extend. What if I need a purple button, or a striped alert box?

Tools instead of widgets.

Come on, how much times did the stock accordion or modal window was fitting exactly your needs out, of the box? Stop wrestling with built-in widgets, and create yor own, tailored on your needs!

FontAwesome

Because FontAwesome is simply... awesome! And now, you'll be able to scale & recolor icons with the lowest effort ever!

About Bootstrap.

Bootstrap is a LESS-powered CSS/JS library to create beautiful web interfaces. Affresco was based on the Bootstrap codebase, ported to the more powerful SCSS language.

Does it even need presentations? Visit the Bootstrap official page .

Affresco is a Bootstrap fork created by Samuele Santi.
